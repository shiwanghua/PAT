# 编程记忆

### 1.scanf && cin

* %lld 对应long型变量

* %d 对应 int

* 获取输入字符

  ```C++
  char c;
  getchar();  // 如果字符前有一个回车或一个空格就用getchar去掉
  scanf(“%c”,&c);
  ```

* 获取输入字符串

  ```c++
  char str[100];
  gets_s(str);// 可以读取空格，回车结束；gets没有了
  int N=strlen(str);
  ```

* 循环获取输入，可直接while(cin>>a>>b){}

### 2. printf && cout

* %02d 输出2位整数，不足则以0填充，%010d, 输出10位整数不足则以0填充
* %.1f 输出一位小数
* int a=100; cout<<fixed<<setprecision(1)<<double(a); // 输出一位小数
* cout填充

```c++
int n=10;char a ='b';
cout<<setw(n)<<setfill(a)<<a;
```

### 4. fill

* 二维数组填充

